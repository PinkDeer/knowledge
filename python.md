### Операторы

| Оператор | Описание |
| ------ | ----------- |
| +   | Сложение |
| - | Вычитание |
| * | Умножение |
| /    | Деление (в результате вещественное число) |
| //    | Деление с округлением вниз |
| **    | Возведение в степень |
| %    | Остаток от деления |

### Заметки

Нижнее подчеркивание вызывает последний полученный результат
```
>>> 7*8
56
>>> _ + 4
60
>>>
```
