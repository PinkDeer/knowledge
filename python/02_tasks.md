Создайте в отдельном файле функцию, переводящую градусы по шкале Цельсия в
градусы по шкале Фаренгейта по формуле: T<sub>F</sub> = 9/5 * T<sub>C</sub> + 32

```
def converter_to_f(cel):
    return 9/5*cel+32

print(converter_to_f(25))
```
